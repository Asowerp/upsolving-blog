# 0307 做题记录

## LeetCode 每日一题

### 题意

给你一个二进制字符串 $s$。你可以按任意顺序执行以下两种操作任意次：

- 类型 1 ：删除 字符串 $s$ 的第一个字符并将它 添加 到字符串结尾。
- 类型 2 ：选择 字符串 $s$ 中任意一个字符并将该字符反转。

请你返回使 $s$ 变成 交替 字符串的前提下， 类型 2 的 最少 操作次数 。

### 思路

一个比较显然的观察是，我们可以转化为 $s+s$ 上的长为 $n:=|s|$ 的滑动窗口。这样就十分简单了，因为交替字符串只有两种情况，分类讨论即可。代码如下：

```cpp
class Solution {
public:
    int minFlips(string s) {
        int n = s.size();
        int cnt1 = 0, cnt2 = 0;
        int ans = 0;
        for (int i = 0; i < n; i++) {
            if (i % 2) {
                cnt1 += s[i] == '1';
                cnt2 += s[i] == '0';
            } else {
                cnt1 += s[i] == '0';
                cnt2 += s[i] == '1';
            }
        }
        if (n % 2 == 0)
            return min(cnt1, cnt2);
        ans = min(cnt1, cnt2);
        for (int i = 0; i < n; i++) {
            if ((i + n) % 2) {
                cnt1 -= s[i] == '0', cnt1 += s[i] == '1';
                cnt2 -= s[i] == '1', cnt2 += s[i] == '0';
            } else {
                cnt1 -= s[i] == '1', cnt1 += s[i] == '0';
                cnt2 -= s[i] == '0', cnt2 += s[i] == '1';
            }
            ans = min({ans, cnt1, cnt2});
        }
        return ans;
    }
};
```
